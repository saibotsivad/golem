Load language model

Loads Xenova/gpt2 — a quantized ONNX export of OpenAI’s GPT-2 (124 M parameters, 50,257-token BPE vocabulary). At ~81 MB it is the largest single asset on this page. Once loaded it is available to §2 (next-token prediction) and §3 (sampling strategies) without re-downloading; files stay in the browser’s transformers-cache between sessions. Contrast with all-MiniLM-L6-v2 (~23 MB, encoder-only, produces fixed-length embeddings) — GPT-2 is a decoder that generates text token-by-token.

console API 
// Check current status before loading
REGISTRY['gpt2-lm']
// → { label: 'GPT-2 LM', size: '~81 MB', status: 'absent' | 'cached', … }

// Load the model (downloads on first visit, reads from cache after)
const m = await golem.loadModel()
// REGISTRY['gpt2-lm'].status → 'ready'

// The model is now shared across all sections via the module-scope lmModel variable.
// §2 and §3 call ensureModel() / golem.loadModel() internally — if you load here
// first, those sections skip the download step entirely.

// Release from memory (browser cache is preserved — next load is instant)
await golem.unloadModel()
// REGISTRY['gpt2-lm'].status → 'cached'
