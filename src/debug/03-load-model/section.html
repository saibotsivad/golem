Load language model

HuggingFace model name for AutoModelForCausalLM.from_pretrained. Try Xenova/distilgpt2 — a distilled GPT-2 (82 M parameters, ~40 MB quantized) that shares GPT-2’s tokenizer and vocabulary; useful for comparing generation quality against the full model at half the weight. For a heavier test: Xenova/gpt2-medium — 345 M parameters, ~170 MB quantized, same tokenizer. Registry keys get a -lm suffix to avoid collision with tokenizer keys (e.g. xenova-distilgpt2-lm). Note: the GPT-2 model used by §2 and §3 is tracked separately as gpt2-lm and loaded via golem.loadModel(), not this form.

console API 
// What's loaded?
golem.models()
// → { 'xenova-distilgpt2-lm': 'ready' }

// Load DistilGPT-2 (same tokenizer as GPT-2, already available as 'xenova-gpt2')
const m = await golem.loadLM('Xenova/distilgpt2')
// REGISTRY key: 'xenova-distilgpt2-lm'

// Run inference — pair with a matching tokenizer
// (DistilGPT-2 uses the same tokenizer as GPT-2)
await golem.loadTokenizer('Xenova/gpt2', false)
const inputs = _loadedTokenizers  // tokenizers are internal; use golem.tokenize() for pieces
// For raw inference, use the returned model directly — same interface as §2:
